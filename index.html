LaTeX Calc

A multi-threaded, realtime LaTeX Computer Algebra System.

Keyboard Shortcuts

  • Shift + Enter: Spawn a new calculation block.
  • Cmd + K (or Ctrl + K): Open the Command Palette (Themes, Settings, Actions).
  • Esc: Close palettes and modals.

Pro Tips

  • Constants: Define global variables (e.g., a = 5) in the left panel to use them globally.
  • Graphing: Type any 1D function to automatically generate an interactive auto-scaling plot.
