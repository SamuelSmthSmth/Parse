LaTeX Calc Architecture

A multi-threaded, realtime LaTeX Computer Algebra System powered by a dual-engine architecture.

1. Browser Engine

Runs standard calculus and algebra directly in your browser using Pyodide and SymPy. It is fast, lightweight, and requires no installation.

2. Desktop Engine (Optional)

For massive matrix determinants, differential equations, and heavy numerical root finding, connect our native Julia companion app for unlimited compute power.

How to install the Desktop Engine

Download Desktop Engine Package (.zip)
  • Step 1: Install Julia from julialang.org
  • Step 2: Extract the downloaded julia-desktop-engine.zip file.
  • Step 3: Launch the engine:
    • Windows: Double-click launcher.bat
    • Mac/Linux: Open terminal, run chmod +x launcher.sh, then ./launcher.sh

Keyboard Shortcuts & Tips

  • Shift + Enter: Spawn a new calculation block.
  • Cmd + K: Open the Command Palette.
  • Constants: Define global variables (e.g., a = 5) in the left panel.
